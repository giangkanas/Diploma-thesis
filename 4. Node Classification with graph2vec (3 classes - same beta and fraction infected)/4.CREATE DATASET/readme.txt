Ουσιαστικά στο createDataset αναζητώ τον καλύτερο συνδιασμό παραμέτρων δημιουργίας node embedding με βάση τα αποτελέσματα σε ένα μοντέλο κατηγοριοποίησης (εδω χρησιμοποιησα το Logistic Regression). Ετσι δημιουργώ τo dataset με βαση το καλυτερο αποτελεσμα προβλεψης.

Απο δω και στο εξής στο επόμενο βημα θα χρησιμοποιηθεί το dataset που δημιουργηθηκε με βαση τα καλύτερα αποτελέσματα
